# Build Directions:
>**C++ Version:** 11.4.0\
>**Build Tool:** Makefile

### Build Steps:
> 1. Open project directory in a terminal
> 2. Compile the project with `make`
> 3. Run the program with `./output` or `make run`
> 4. Run `make clean` to clean up compiled files

### Debug Mode:
To enter debug mode, type `m`, and type it again to exit
